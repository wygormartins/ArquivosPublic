# ArquivosPublic

Repositório público — imagens, animações e paleta de cores, acessíveis via URL direta para uso em dashboards e ferramentas internas.

**URL base:**

```text
https://raw.githubusercontent.com/wygormartins/ArquivosPublic/main/
```

---

## Animações Lottie

Arquivos JSON para uso com [lottie.link](https://lottielink.com) ou qualquer player Lottie.

**Formato da URL:**

```text
https://raw.githubusercontent.com/wygormartins/ArquivosPublic/main/LottiesJson/{arquivo}.json
```
